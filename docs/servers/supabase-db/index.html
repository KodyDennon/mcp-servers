Get Started

Installation Options

Choose the installation method that fits your workflow

01

Global CLI Installation

Install globally for system-wide access via command line

npm install -g mcp-supabase-db

# Start the server
supabase-db-mcp

# Or with custom root path
MCP_SUPABASE_ROOT=/path/to/config supabase-db-mcp
02

Project-Local Installation

Install in your project for version-controlled dependencies

npm install mcp-supabase-db

# Run with npx
npx mcp-supabase-db

# Or add to package.json scripts
"scripts": {
  "mcp": "mcp-supabase-db"
}
03

Monorepo Development

Work directly in the monorepo for contributing or customization

git clone https://github.com/KodyDennon/mcp-servers.git
cd mcp-servers
pnpm install

# Build and start
pnpm --filter mcp-supabase-db build
pnpm --filter mcp-supabase-db start

Execution

Dual Execution Modes

Choose between direct MCP tools or code execution based on your needs

‚ö°

Direct Tools Mode

Default mode with 35 specialized MCP tools for database operations. Provides precise control and explicit tool invocations. 

# Default mode
supabase-db-mcp

# Or explicitly set
MCP_MODE=tools supabase-db-mcp
üîí

Code Execution: Sandbox

Isolated execution environment for enhanced security. Operations run in a sandboxed container with limited system access. 

MCP_MODE=code-api \
CODE_EXECUTION_MODE=sandbox \
supabase-db-mcp
üöÄ

Code Execution: Direct

Full system access for maximum flexibility. Use when you trust the AI agent completely and need unrestricted operations. 

MCP_MODE=code-api \
CODE_EXECUTION_MODE=direct \
supabase-db-mcp
Capabilities

Feature Highlights

Everything you need for production database operations

Schema Management

Introspect database schemas, tables, columns, indexes, and constraints. Create, modify, and drop database objects with full PostgreSQL DDL support.

Data Operations

Execute queries, inserts, updates, and deletes. Stream large result sets, handle transactions, and work with JSON/JSONB data types efficiently.

Migration System

Generate, apply, and rollback database migrations. Track migration history and maintain schema version control across environments.

Auth Management

Manage Supabase authentication users, roles, and policies. Create RLS policies, manage permissions, and handle user operations.

Storage Operations

Upload, download, and manage files in Supabase Storage. Create buckets, set policies, and handle large file uploads with resumable transfers.

AI-Powered Indexing

Generate embeddings for text data, index directories and URLs for RAG, and perform semantic search with pgvector integration.

Real-time Subscriptions

Set up real-time database subscriptions, listen for changes, and handle events from PostgreSQL triggers and Supabase Realtime.

Smart Caching

Intelligent result caching reduces latency and costs. Schema introspection and frequent queries are cached with configurable TTLs.

Privacy Filters

Built-in privacy filters prevent accidental exposure of sensitive data. Configurable filters for PII, credentials, and custom patterns.

Streaming Results

Stream large query results without blocking. Handle datasets exceeding token limits with progressive result delivery.

Edge Functions

Deploy and manage Supabase Edge Functions. Invoke functions, view logs, and handle serverless operations directly from your AI agent.
