Quick Start Guide

Three simple steps to connect your AI agents to Supabase

01

Install the Server

Install globally via npm or use npx for one-time execution

npm install -g mcp-supabase-db
# or use npx
npx mcp-supabase-db
02

Configure Environment

Set up your Supabase credentials and connection details

POSTGRES_URL_NON_POOLING="postgresql://..."
SUPABASE_URL="https://xxx.supabase.co"
SUPABASE_SERVICE_ROLE_KEY="your-key"
OPENAI_API_KEY="sk-..."
03

Connect Your AI Tool

Add the server to Claude Desktop, Cursor, or your MCP client
